# Gifts & Flowers Shop

A beautiful ecommerce platform for gifts and flowers.

Demo: https://giftsandflowers.onrender.com/


How to run:

<br>

create .env file

add your jwt secret key

add your mongodb_uri


<br><br>
open terminal..

cd frontend

npm i

npm start


<br><br>
open new terminal..

cd backend 

npm i

npm start
